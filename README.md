# ローカル環境構築方法

```
pip install -r requirements.txt

<readthedocs-test-mkdocsのルートディレクトリに移動>

mkdocs serve
```

# ドキュメント作成方法

`docs`配下にmdファイル作るとページ作れます

- ディレクトリ構造も作れます
- ファイルの変更を保存すれば、ホットリロードが走って自動でブラウザのページに反映されます
